OSINOS

Lightweight Linux UI for Windows

About the Project

OSINOS is a lightweight, application-layer Linux environment running on Windows, inspired by the futuristic Edex UI, designed to provide a clean and powerful interface for your everyday needs.

Features

  • Terminal-like UI with a modern, minimalistic design.
  • Real-time system stats such as CPU, RAM, and network usage.
  • Interactive, smooth animations that make your workflow intuitive.
  • Fully customizable desktop environment with a futuristic look and feel.

Installation
